Available API Endpoints

Event APIs

  • GET /api/v1/events/:id - Get event details
  • POST /api/v1/events/:id/register - Register for event
  • POST /api/v1/events/:id/deregister - Deregister from event
  • POST /api/v1/events/:id/pay - Pay for event

Auth APIs

  • POST /api/v1/auth/otp/send - Send OTP
  • POST /api/v1/auth/otp/verify - Verify OTP
  • GET /api/v1/auth/me - Get current user

User APIs

  • GET /api/v1/users/me - Get user profile
  • PUT /api/v1/users/me - Update user profile

Backend Setup Required

To test API connectivity, ensure the Go backend server is running:

  1. Navigate to the backend directory
  2. Start the server: make watch
  3. Server should be running on port 8000
  4. Return to this page and test with a real event ID
